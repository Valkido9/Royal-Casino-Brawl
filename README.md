&#x20;🎲 孟加拉国皇家赌场 - 任地狱明星大乱斗



这是一个基于 Pygame 开发的群友斗蛐蛐游戏。里面充满了群友，持续更新中。

参考了SpeenTF2的小球游戏和明日方舟的斗蛐蛐。



&#x20;🚀 如何运行



1\. 克隆本项目：

&#x20;  `git clone https://github.com/你的用户名/你的仓库名.git`

2\. 进入目录：

&#x20;  `cd 你的仓库名`

3\. 安装依赖库：

&#x20;  `pip install -r requirements.txt`

4\. 运行游戏：

&#x20;  `python main.py`
